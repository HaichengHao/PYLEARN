|  os模块的相关函数 |  说明 |  
| ------ | ----|
| os.getcwd()  | 返回当前的工作目录   |
|  os.listdir() | 返回指定路径下的文件和目录信息   |
| os.mkdir(path[,mode])  | 创建目录   |
|  os.makedirs(path1/path2... [,mode]) | 创建多级目录   |
| os.rmdir(path)  | 删除目录   |
| os.removedirs(path1/path2...)  | 删除多级目录   |
| os.chdir(path) |  将path（自定义就行）设置为当前工作目录 |
